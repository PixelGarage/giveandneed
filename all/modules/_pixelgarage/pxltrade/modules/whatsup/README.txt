WhatsUp module
==============
This module can register any phone number on the WhatsApp platform and use it to send messages via WhatsApp.

This implementation depends heavily on the whatsapp/chat-api by WHAnonymous.

Documentation
=============
Docu about the Chat-API can be found here: https://github.com/WHAnonymous/Chat-API/wiki
